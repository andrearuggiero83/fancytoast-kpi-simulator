===============================
Fancytoast KPI Simulator Build Kit
===============================

▶ FOR WINDOWS (.exe):
---------------------
1. Make sure Python and pyinstaller are installed:
   pip install pyinstaller

2. Run in terminal:
   pyinstaller fancytoast.spec

3. Output .exe will be in /dist

▶ FOR MAC (.app):
---------------------
1. Make sure Python and py2app are installed:
   pip install py2app

2. Run in terminal:
   python setup.py py2app

3. Output .app will be in /dist

Enjoy the Fancytoast KPI Simulator!
